# 花火撮影レンズ計算ツール

花火撮影に最適なレンズmm数を計算し、構図をプレビューできるWebアプリケーションです。

## 機能

- **レンズmm数計算**: 撮影地から花火までの距離を入力すると、最適なレンズmm数を自動計算
- **構図プレビュー**: リアルタイムで撮影イメージを視覚的に確認
- **地上の割合調整**: 地上部分の割合を調整して構図を最適化
- **センサーサイズ対応**: フルサイズ、APS-C、マイクロフォーサーズに対応

## 使い方

1. `index.html`をブラウザで開く
2. 撮影地から花火までの距離（m）を入力
3. 地上の割合（%）を調整して構図を決定
4. センサーサイズを選択（デフォルト: フルサイズ）
5. 花火の高さを設定（一般的には200-400m）
6. 推奨レンズmm数が表示され、プレビューで構図を確認

## 計算式

レンズmm数は以下の式で計算されます：

```
画角 = 2 × arctan(花火の高さ / (2 × 距離))
実効画角 = 画角 / (空の割合 / 100)
レンズmm = (センサー高さ / 2) / tan(実効画角 / 2)
```
